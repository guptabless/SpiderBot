# SpiderBot
This script crawls the links from the webpages provided by you.
## Requirment:

### packages 

- requests
- bs4
- argparse
- sys
- re
- bcolors

### python > 3.x 

## usage: 

SpiderBot.py  -c < URL which you want to search > 


OPTIONS: 

```
-h           --help    
                	< show the available options >
-c             valid URL with https:// or http://
  		            < URL which links you want to search >

```
